Installing fun-bots is like installing any other mod for Venice Unleashed.

Requirements

Fun-bots require a working Venice Unleashed server.

Software requirements

There are no special software requirements for fun-bots.

Fun-bots require at least VeniceEXT 1.1.0, this is automatically installed by the server software.

Hardware requirements

There are no special hardware requirements for fun-bots other than the Venice Unleashed server requirements.

Manual Setup

TBA

1. Download the best version for fun-bots.

Pick the most appropriate version for your server. We recommend “Full Release” or “Release Candidates” on production server and “Development Releases” for development-related builds. 
 

2. Unpack the compressed file

Unpack the content of the downloaded file to your Admin\Mods directory. (Usually located at My Documents\Battlefield 3\Server\Admin\Mods). Verify that the files inside are correct (it should contain the mod.json, etc).

  • Due to a bug with Venice Unleashed, it is highly recommended to rename funbots-1.0.0.0 to simply funbots to make console commands to work.

 

3. Add the name of the mod to the ModList.txt

Create a new file in Admin folder, only if it doesn't exist yet, and name it ModList.txt.

Edit the file and add, or append the name of the fun-bots mod folder to the ModList.

4. Start up the server

You can now start up your server and it should start a short installation process where it'll install the latest content.
