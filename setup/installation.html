Fun-bots is a mod for Venice Unleashed and requires a working Venice Unleashed server.

Installation

The installation is extremely simple.

1. Download the best version for fun-bots.

 

2. Unpack the compressed file.

Unpack the content of the downloaded file to your Admin\Mods directory. (Usually located at My Documents\Battlefield 3\Server\Admin\Mods).

  • Due to a bug with Venice Unleashed, it is highly recommended to rename funbots-1.0.0.0 to simply funbots

 

3. Add the name of the mod to the ModList.txt

Create a new file in Admin folder, only if it doesn't exist yet, and name it ModList.txt.

Edit the file and add, or append the name of the fun-bots mod folder to the ModList.

4. Start up the server

You can now start up your server and it should start a short installation process where it'll install the latest content.
